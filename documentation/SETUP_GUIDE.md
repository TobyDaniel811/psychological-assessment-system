# Psychological Assessment System — Setup & Run Guide

A web-based behavioural survey and psychological assessment system using
a Random Forest classifier trained on real survey data.

---

## 1. Requirements

- Python 3.10 or newer
- pip

---

## 2. Install dependencies

From the project root:

```bash
pip install -r requirements.txt
```

---

## 3. Train the model

This reads `dataset/Psychological_Assessment_Dataset.xlsx`, cleans it,
encodes it, trains the Random Forest, evaluates it, and saves everything
the Flask app needs into `model/`.

```bash
python train_model.py
```

You should see console output ending in:

```
Saved: model/random_forest_model.pkl
Saved: model/feature_columns.pkl
Saved: model/ordinal_maps.pkl
Saved: model/nominal_cols.pkl
Training pipeline complete.
```

A confusion matrix image is also saved to `documentation/confusion_matrix.png`.

This step also requires one more small artifact used by the survey form
(the list of valid nominal answer options). Run:

```bash
python -c "
import joblib, pandas as pd
nominal_cols = joblib.load('model/nominal_cols.pkl')
df = pd.read_excel('dataset/Psychological_Assessment_Dataset.xlsx')
nominal_options = {col: sorted(df[col].unique()) for col in nominal_cols}
joblib.dump(nominal_options, 'model/nominal_options.pkl')
print('Saved model/nominal_options.pkl')
"
```

> Both of the above only need to be run once, or whenever the dataset changes.

---

## 4. Create an administrator account

Admin accounts cannot be created through the public registration form
(by design — this is a security decision, not an oversight). Run:

```bash
python create_admin.py
```

You will be prompted for a username, email, and password.

---

## 5. Run the application

```bash
python -m flask --app app.app run --debug
```

Then open **http://127.0.0.1:5000** in your browser.

---

## 6. Using the system

| Page | URL | Access |
|---|---|---|
| Home | `/` | Public |
| Register | `/register` | Public |
| Login | `/login` | Public |
| Survey | `/survey` | Logged-in users |
| Result | `/result/<id>` | Owner only |
| History | `/history` | Logged-in users |
| Admin Dashboard | `/admin` | Admins only |

---

## 8. Retraining with new or updated data

If `dataset/Psychological_Assessment_Dataset.xlsx` changes, re-run:

```bash
python train_model.py
```

This regenerates everything inside `model/` automatically. No changes to
`app/` are ever required for a retrain — the Flask app always loads
whatever currently sits in `model/`.

---

## 9. Important note for the project report

The system displays the model's prediction as a behavioural/psychological
screening result, not a clinical diagnosis. The result page includes an
explicit disclaimer recommending professional consultation. This framing
should be preserved in any further UI changes.
